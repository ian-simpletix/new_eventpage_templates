## Viewing Templates

Once GitHub Pages is enabled, templates will be available at:
```
https://ian-simpletix.github.io/new_eventpage_templates/Claude Adjusted/{category}/{template}.html
```

## Features

- All templates use relative paths: `../../images/filename.png`
- Mobile responsive with sticky CTAs
- No external dependencies (Tailwind loaded via CDN where needed)
- Ready to deploy to S3 or any static hosting
